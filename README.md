# movie-api

Post /metadata <br/>
https://localhost:44389/metadata

Get /metadata/:movieId <br/>
https://localhost:44389/metadata/3

Get /movies/stats <br/>
https://localhost:44389/movies/stats


## Using

- .NET Core 3.1<br/>
- CSVHelper 27.1.1

###  Install
For .NET CLI to install CSVHelper<br/>
dotnet add package CsvHelper --version 27.1.1
